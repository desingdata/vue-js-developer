1 · Template inline & Sintaxis

El template se declara dentro del componente (opción template). La sintaxis {{ }} evalúa expresiones JavaScript en el HTML.

Hoy es: {{ new Date().toLocaleDateString('es-CO') }}

Estado de la app: {{ darkMode ? 'Modo oscuro activo' : 'Modo claro activo' }}

2 · Directivas & Atributos para propiedades
